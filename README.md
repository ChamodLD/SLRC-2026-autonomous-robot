# 🏆 SLRC 2026 – Autonomous Vision-Based Robotic System  
### Team BUMBLEBEE

> Autonomous mobile robotic system developed for the **Sri Lanka Robotics Challenge (SLRC 2026)**.

---

## 📖 Overview

The **SLRC 2026 Autonomous Vision-Based Robotic System** is an intelligent mobile robot designed to perform fully autonomous navigation and task execution in a competition environment.

The system integrates **computer vision, machine learning, and embedded control systems** to perceive its surroundings, make decisions in real time, and execute tasks without human intervention.

---

## ✨ Features

- Autonomous navigation and task execution  
- Real-time object and shape detection  
- Vision-based object detection and distance estimation  
- Custom-trained **YOLOv8** object detection model  
- Raspberry Pi–based perception and decision-making system  
- Embedded control integration with microcontrollers  

---

## 🧠 Technologies Used

### Hardware
- Raspberry Pi 4
- Raspberry Pi Camera Module V2
- Arduino-based control system
- SolidWorks mechanical design

### Software & AI
- Python
- OpenCV
- PyTorch
- YOLOv8
- Roboflow
- Arduino IDE

---

## 🏗 System Overview

The system consists of:

- **Vision Module (Raspberry Pi)** → Object detection & environment perception  
- **AI Model (YOLOv8)** → Object classification and detection  
- **Control System (Arduino)** → Motor control and actuation  
- **Mechanical System** → Robot chassis and mobility design
